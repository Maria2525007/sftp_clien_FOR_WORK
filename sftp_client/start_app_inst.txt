Инструкция по работе с приложением

1) Я использовала приложение на тестовом сервере sftp_user@localhost, подключаясь к нему через wsl Ubuntu
В конфигурационном файле /etc/ssh/sshd_config необходимо открыть доступ к подключению
Добавив строку Subsystem sftp internal-sftp

Команда для перезапуска сервера sudo service ssh restart

2) Подключиться к серверу по sftp -P 22 sftp_user@172.19.60.50
